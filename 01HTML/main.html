se necesita un desarrollo web mediante el cual se pueda diseñar una serie de cuestionarios para la evaluacion de rendimiento y productividad, calidad, procesos administrativos y una evaluacion general sobre los docentes de la UTC pertenecientes a la carrera de isc. El sistema debe contar con una seccion para la creacion de preguntas CRUD, asimismo debe de tener las opciones de la creacion de los formularios: 1.- Pregunta SI NO 2.- Opcion multiple en base a una sola respuesta/multiple respuesta 3.- Preguntas con base a una escala LICKER donde se ingrese el numero de respuestas contra elementos de la escala 4.- El sistema debe de contar con el guardado e historial de la evaluacion por cada docente, cuatrimestre, asignatura 5.- Una primera etapa, la evaluacion sera 100% anonima, brindando una opcion de seguridad para que la evaluacion sea realizada una sola vez por ciclo. 6.- La evaluacion se realiza por parte de personal administrativo y comunidad estudiantil 7.- Segunda etapa, se debe de contar con un inicio de sesion autenticado por medio de la API de Office 365 para todo personal de UTC 8.- Debe tener la logica de separacion para la seccion administrativa y estudiantil -Reglas de estilo (Viernes 3)

Cada equipo debe subdividir los roles Programador (Validacion de las reglas), Diseñador(Valida el diseño), Lider(Valida que todo funcione). Todos harán de todo "SEGUN" Primer entregable es la arquitectura, de informacion como del sistema de red Que se diseñe todo el cascaron
